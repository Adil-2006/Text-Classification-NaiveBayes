# Text Classification Using Naive Bayes

## Project Overview

This project implements a Text Classification system using the Multinomial Naive Bayes algorithm. The model classifies SMS messages as either Spam or Ham (Not Spam).

## Dataset

SMS Spam Collection Dataset

* Total Messages: 5572
* Ham Messages: 4825
* Spam Messages: 747

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* NLTK
* Matplotlib
* Seaborn

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Text Preprocessing
4. TF-IDF Feature Extraction
5. Train-Test Split
6. Naive Bayes Training
7. Model Evaluation
8. Custom Prediction

## Evaluation Metrics

* Accuracy: 97.49%
* Precision: 100%
* Recall: 81.33%
* F1 Score: 89.71%

## Outputs

* Confusion Matrix
* Trained Model (.pkl)
* TF-IDF Vectorizer (.pkl)

## How to Run

```bash
pip install -r requirements.txt
python main.py
```
